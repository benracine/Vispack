About {{ SITE_TITLE }}

{{ SITE_TITLE }} solves the problem in the {{ FRAMEWORK_TITLE }} community of being able to easily identify good apps, frameworks, and packages. Ever want to know which is the most popular or well supported {{ FRAMEWORK_TITLE }} blog, content management system, or api tool? {{ SITE_TITLE }} solves that problem for you!

What is a {{ SITE_TITLE }}?

A {{ SITE_TITLE }} is anything that is involved in the {{ FRAMEWORK_TITLE }} ecosphere that can be stored on a repository such as Github or Bitbucket. If it can be stored in PyPI, that's even better!

When did you code this?

Open Comparison was conceived for and coding during Django Dash 2010. Since then we've been feverishly adding more features, improving the design, and trying to make data entry as easy/intuitive as possible.

Who built this thing?

Daniel Greenfeld and Audrey Roy are the creators of this project. Additional contributing members are detailed in the team page.

Can I help?

Open Comparison is an volunteer based open source effort. Please submit any ideas or issues and also fork and submit back patches and enhancements.
