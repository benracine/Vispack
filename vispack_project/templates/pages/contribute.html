How can you contribute?

As open source projects, {{ SITE_TITLE }} and Open Comparison provides several ways to further the progress of this site:

Contributing code

This project has had over 30 contributors whose work has seen this project move forward in leaps and bounds. If you want help out, please read the contributing instructions. You should also read the Installation Section of the Open Comparison FAQ.

Contributing financially

As much as the leaders of Open Comparison love to work on this project, as hourly consultants financial considerations often get in the way. While a PayPal button has been considered, other options are being offered at this time:

Post jobs on Whitespace Jobs

The team behind this site and Open Comparison have launched a job listing site for developers called Whitespace Jobs.

Out of every paid listing, $100 will go towards work on Open Comparison or to diversity advocacy groups such PyLadies.

Direct Open Comparison Consulting

The project leaders are open to the idea of doing consulting work to enhance and modify the site and the . All produced code (except passwords, API keys, and other security issues) must remain BSD license compliant. If you are interested, please contact pydanny or audreyr.
